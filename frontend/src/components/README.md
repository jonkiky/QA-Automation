# Components Directory

This directory contains reusable React components for the QA Automation platform.

## Structure

- `ui/` - Base UI components (buttons, cards, tables, etc.)
- `charts/` - Chart components using Recharts
- `flows/` - Traceability flow components using React Flow
- `layouts/` - Layout components (headers, sidebars, etc.)
- `features/` - Feature-specific components organized by domain
